# Daily Ways 📅

A personal project to track and manage daily routines, habits, and productive ways of living.

## Overview

Daily Ways is designed to help you:
- 📋 Track daily tasks and habits
- ✅ Monitor progress on personal goals
- 🎯 Build consistent routines
- 📊 Analyze productivity patterns

## Project Structure

```
daily-ways/
├── README.md
├── habits/
│   └── daily_habits.md
├── routines/
│   ├── morning_routine.md
│   └── evening_routine.md
└── tracking/
    └── weekly_summary.md
```

## Daily Habits to Track

- [ ] Morning meditation (15 min)
- [ ] Exercise (30 min)
- [ ] Reading (20 min)
- [ ] Learning new skill (1 hour)
- [ ] Journaling (10 min)

## Getting Started

1. Clone this repository
2. Add your personal daily habits in `habits/daily_habits.md`
3. Create your routines in the `routines/` folder
4. Track weekly progress in `tracking/weekly_summary.md`
